title: 如何使用 Gemini CLI
date: 2025-08-05 16:13:47
---

Gemini CLI 是一款功能强大的开源人工智能助手，可直接在您的终端（命令行界面）中使用。 您可以透过自然语言与它互动，以协助您完成编码、文件分析、错误修复等任务。

### 1. 安装准备

在使用 Gemini CLI 之前，您需要确保您的电脑已安装 **Node.js**（版本 18 或更高）。

*   **检查 Node.js 版本：**
    打开您的终端（在 Windows 上是 PowerShell 或命令提示符，在 Mac 上是 Terminal），然后输入以下命令：
    ```bash
    node -v
    ```
*   **安装 Node.js：**
    如果未安装或版本过低，请前往 [Node.js 官网](https://nodejs.org/) 下载并安装 "LTS" (长期支持) 版本。

### 2. 安装 Gemini CLI

您可以使用 `npm`（Node.js 的包管理器）来安装 Gemini CLI。建议进行全局安装，以便在任何目录下都能轻松启动。

*   **全局安装命令：**
    在终端中执行以下命令：
    ```bash
    npm install -g @google/gemini-cli
    ```

### 3. 首次运行与授权

安装完成后，在终端中输入 `gemini` 并按回车即可启动。

*   **首次运行设置：**
    1.  **选择主题：** 程序会提示您为界面选择一个喜欢的颜色主题。
    2.  **Google 账号授权：** 接着，它会引导您在浏览器中登录您的 Google 账号。 这是为了授权 Gemini CLI 使用 Gemini 1.5 Pro 模型。此过程是安全的。

完成以上步骤后，您就可以开始与 Gemini CLI 进行互动了。免费用户每分钟最多可发送 60 个请求，每天最多 1000 个请求，足以满足日常开发需求。

### 4. 基本用法

进入 Gemini CLI 的交互界面后，您可以直接输入自然语言提示来执行任务。

*   **代码相关任务：**
    *   `请帮我用 Python 写一个快速排序算法。`
    *   `解释一下这个文件里的代码是做什么的？ @文件名`
    *   `找出项目中的所有 TODO 注释。`
*   **文件操作：**
    Gemini CLI 可以读取和修改您项目中的文件。在执行修改前，它会预览变更并请求您的批准。
*   **执行 Shell 命令：**
    您可以在 Gemini CLI 中直接执行终端命令，只需在命令前加上 `!`。
    ```
    !ls -l
    ```
*   **其他命令：**
    *   `/help`: 显示帮助信息和可用命令。
    *   `/quit`: 退出 Gemini CLI。
    *   `/theme`: 更改界面主题。
